# Alpaca Generator

An interactive web app that lets you design and customize your own alpaca!  
Users can randomize features, accessorize with different styles, and download their creations as images.

## Overview
The Alpaca Generator is a fun front‑end project built with **HTML**, **CSS**, and **JavaScript**.  
It provides a simple UI where users can:
- Generate a random alpaca with unique features.
- Select specific categories (e.g., accessories, backgrounds).
- Choose style variants for each category.
- Download the final alpaca as an image.
